# Vengadesh Raj Portfolio

Cyberpunk-themed static portfolio for Vengadesh Raj, generated from the supplied resume.

## Run Locally

Open `index.html` directly in a browser

## Structure

- `index.html` - page content
- `styles.css` - responsive cyberpunk visual system
- `script.js` - mobile nav, active section state, and ambient canvas animation
- `assets/hero-cyberpunk.png` - generated hero background
- `assets/vengadesh-raj-resume.pdf` - downloadable resume
